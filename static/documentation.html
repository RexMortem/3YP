YAPPL

Yet Another Probabilistic Programming Language

A simple interpreted language for declaring variables, performing arithmetic, and working with probability distributions.

Language Reference
Statements
Syntax Description
let x;Declare a variable (initialised to 0)
let x = expr;Declare and assign
x = expr;Reassign an existing variable
output(expr);Print a value to the output panel
Expressions & Operators
SyntaxDescription
42, 3.14Integer and float literals
xVariable reference
a + b, a - bAddition, subtraction
a * b, a / bMultiplication, division
-xNegation
d1 + d2Combine two distributions (convolution of outcomes)
Distributions
SyntaxDescription
uniform(a, b) Discrete uniform distribution over integers [a, b]
uniformContinuous(a, b) Continuous uniform distribution over [a, b]
Discrete(v1:p1, v2:p2, ...) Custom discrete distribution with explicit value:probability pairs
Distribution Methods
SyntaxDescription
d:expect(v)P(X = v) — probability of a specific outcome
d:min()Lower bound of the distribution
d:max()Upper bound of the distribution
d:mean()Expected value (mean) of the distribution
