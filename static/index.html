YAPPL — Yet Another Probabilistic Programming Language

A simple language for declaring variables and working with probability distributions.

Basic Syntax 
let x = 5;
x = x + 1;
output(x);
Distributions 
let d = uniform(1, 6);
let c = uniformContinuous(0, 1);
let k = Discrete(1:0.3, 2:0.7);
let s = d1 + d2;
Distribution Methods
  • d:expect(v) — probability that X = v
  • d:min() — lower bound
  • d:max() — upper bound
  • d:mean() — expected value
Statements
  • let x; — declare variable
  • let x = expr; — declare & assign
  • x = expr; — reassign
  • output(expr); — print value
